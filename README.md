android_scripts
===============

Helpful scripts for building Carbon

QuickStart:

    git clone https://github.com/Matoxxi/hudson.git -b kk carbon;
    cd carbon;
    repo init -u https://github.com/CarbonDev/android.git -b kk;
    ./Update.sh;
    ./CompileROM.sh -u; (Xperia U)
